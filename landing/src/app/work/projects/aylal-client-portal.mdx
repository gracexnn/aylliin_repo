## Тойм

Аялал Клиент Портал нь Монголын аяллын салбарт зориулан боловсруулсан орчин үеийн вэб апп юм.
Үйлчлүүлэгчид аяллын багцуудыг хайж, харьцуулж, захиалга хийж, онлайнаар төлбөр тооцоо хийх
боломжтой. Бүх мэдээлэл Admin Порталаас бодит цагт шинэчлэгддэг тул хэрэглэгч үргэлж
хамгийн сүүлийн мэдээлэлтэй ажилладаг.

## Гол боломжууд

- **Ухаалаг хайлт**: Очих газар, огноо, хүний тоо, үнийн хязгаар болон бусад шүүлтүүрээр аяллын
  багцыг олох боломж. Хайлтын үр дүн бодит цагт шинэчлэгддэг.
- **Аяллын дэлгэрэнгүй**: Хөтөлбөр, зураг, газрын зураг, хамрагдах үйлчилгээний жагсаалт
  болон үнийн нарийвчилсан задаргаа бүхий дэлгэрэнгүй хуудас.
- **Онлайн захиалга**: Зочдын мэдээлэл оруулах, баримт бичиг хавсаргах, захиалгын баталгаажуулалт
  авах бүрэн дижитал захиалгын урсгал.
- **Аяллын менежмент**: Хувийн хэрэглэгчийн профайл дахь захиалгын түүх, ирэх аялал, цуцлалтын
  менежмент болон pdf баримт татаж авах.

## Ашигласан технологи

- **Next.js 16**: Server-side rendering болон App Router ашиглан хурдан, SEO-д ээлтэй апп.
- **TypeScript**: Кодын найдвартай байдлыг хангасан хатуу төрлийн систем.
- **Tailwind CSS**: Responsive, орчин үеийн UI загварчлал.
- **Axios**: Backend API-тай холбоо тогтоох HTTP клиент.
- **Framer Motion**: Хэрэглэгчийн туршлагыг сайжруулах анимейшн болон шилжилтүүд.

## Хөгжүүлэлтийн сорилтууд

Гол сорилт нь бодит цагт мэдээлэл синхрончлох явдал байсан – аяллын ирлэг тоо байнга өөрчлөгддөг
тул хэрэглэгчид хамгийн шинэ мэдээллийг харах хэрэгтэй байдаг. Үүнийг шийдэхийн тулд
оновчтой кэшинг болон серверийн нэмэгдлийн шинэчлэлтийн механизм нэвтрүүлсэн.

## Үр дүн

Системийг туршилтаар ажиллуулсны дараа захиалгын урсгал 60%-иар хурдассан бөгөөд
үйлчлүүлэгчдийн хэрэглэгчийн туршлагын үнэлгээ мэдэгдэхүйц сайжирсан. Системийн
хариу өгөх хугацаа дунджаар 1.2 секундэд хүрсэн.
